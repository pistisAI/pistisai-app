# Documentation Archive

This directory is for historical documents that are useful for context but are not current operating guidance.

Use this archive for:

- Completed implementation plans.
- Generated implementation summaries.
- One-off stabilization reports.
- Design notes superseded by architecture docs.
- Vendor, deployment, or workflow experiments that are no longer the primary path.

Do not use archived files as a source of truth without checking current code and canonical docs first.
