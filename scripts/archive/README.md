# Scripts Archive

This directory contains historical scripts that are no longer part of the active deployment workflow but are preserved for reference.

## Contents

| Script | Description |
|--------|-------------|
| `build_and_package.sh` | Legacy build and packaging script |
| `complete_automated_deployment.sh` | Full automated deployment (superseded by current CI/CD) |
| `deploy_to_vps.sh` | Direct VPS deployment script |
| `run_virtual.sh` | Local virtual environment runner |

## Usage

Do not use these scripts for production deployments. Refer to the current deployment documentation in `DEPLOYMENT.md` and `docker-compose*.yml` files for active workflows.

These scripts are preserved for:
- Understanding historical deployment approaches
- Reference when debugging legacy infrastructure
- Context for migration decisions
